## Introduction
<p>
This is an api endpoint to provide data for display chart for Angular frontend and Electron project.
</p>

<p>
This system is for api backend side. Please go to this repository for the frontend side 
<pre> https://github.com/atikahadzman/angular-frontend-test.git </pre> 
or 
<pre>https://github.com/atikahadzman/electron-app.git </pre>
</p>


## Configuration

After clone this repository, you may need to run this command

```bash
cd backend-chart
```

```bash
npm install
```


## Install Express library 

Install Express, which is the most popular library used to build web servers and APIs:

```bash
npm install express
```


## Run the server

Run this command to run in the browser

```bash
node server.js
```
